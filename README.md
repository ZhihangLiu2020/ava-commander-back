# ava-commander-back
- 使用koa2框架搭建的服务端，是ava-commander-front的配套项目
- 参考了博客[vue+koa2+mysql前后端访问](https://blog.csdn.net/qq_40323256/article/details/109261939)
- 启动
  - npm install
  - npm run dev
